  • Comunidades: Número ilimitada de comunidades. Desglose de fincas por bloques o portales. Permite hasta tres cuentas bancarias ordenantes y dos más de orden interno.

 

 

  • Contabilidad: El usuario no requiere conocimientos contables. Olvídese de arduas labores de contabilización. La mayoría de las operaciones contables se generan de forma automática y transparente para el usuario.

 

 

  • Facturación: Total flexibilidad para generar las remesas periódicas a partir de las cuotas fijas y variables seleccionadas previamente. También permite incorporar en la facturación la compensación de saldos impagados. Se generan los adeudos bancarios adaptados a la nueva normativa SEPA.

 

  • Lectura de contadores: Se dispone de una utilidad muy potente para la grabación masiva de lecturas que proporciona al usuario una gran agilidad y facilidad para la entrada de datos.

 

 

  • Presupuestos: A partir de la liquidación o cierre del ejercicio anterior se generan las partidas de gastos del presupuesto del año siguiente para que el administrador solo tenga que retocarlo. También permite incorporar automáticamente las nuevas cuotas periódicas a las fincas derivadas del nuevo presupuesto.

 

  • Informes: Se dispone de un amplio catalogo de listados y resúmenes que se visualizan previamente en pantalla y a continuación podrá enviarlo a la impresora o guardar a un archivo en disco, pudiendo escoger entre diferentes tipos de formatos (Pdf, Word, Excel, etc.), lo que supone una perfecta integración con otras herramientas ofimaticas.

 

 

  • Juntas: Dispone de un Parte de propiedades en el que se figura para cada propiedad, entre otros aspectos, su coeficiente de propiedad y si tiene derecho al voto.

 

  • Incidencias: Gestor de incidencias por comunidades. Desde esta utilidad podrá realizar un seguimiento del estado de las incidencias asociado a cada comunidad.
